# 项目结构说明

##  主体结构

```
src/
├── api/               # API接口
├── assets/            # 静态资源
│   ├── icons/         # 图标资源
│   └── images/        # 图片资源
├── components/        # 公共组件
├── directive/         # 自定义指令
├── layout/            # 布局组件
├── router/            # 路由配置
├── store/             # 状态管理
├── styles/            # 全局样式
├── utils/             # 工具函数
└── views/             # 页面组件

```
